



# angular.isDate








Determines if a value is a date.







  

## Usage
```js
angular.isDate(value);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| value | * | <p>Reference to check.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| boolean | <p>True if <code>value</code> is a <code>Date</code>.</p>  |
