



# angular.isUndefined








Determines if a reference is undefined.







  

## Usage
```js
angular.isUndefined(value);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| value | * | <p>Reference to check.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| boolean | <p>True if <code>value</code> is undefined.</p>  |
