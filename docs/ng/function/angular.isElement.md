



# angular.isElement








Determines if a reference is a DOM element (or wrapped jQuery element).







  

## Usage
```js
angular.isElement(value);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| value | * | <p>Reference to check.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| boolean | <p>True if <code>value</code> is a DOM element (or wrapped jQuery element).</p>  |
