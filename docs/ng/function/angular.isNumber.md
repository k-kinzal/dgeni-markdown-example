



# angular.isNumber








Determines if a reference is a `Number`.







  

## Usage
```js
angular.isNumber(value);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| value | * | <p>Reference to check.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| boolean | <p>True if <code>value</code> is a <code>Number</code>.</p>  |
