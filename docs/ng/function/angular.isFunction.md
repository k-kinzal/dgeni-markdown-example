



# angular.isFunction








Determines if a reference is a `Function`.







  

## Usage
```js
angular.isFunction(value);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| value | * | <p>Reference to check.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| boolean | <p>True if <code>value</code> is a <code>Function</code>.</p>  |
