



# angular.isArray








Determines if a reference is an `Array`.







  

## Usage
```js
angular.isArray(value);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| value | * | <p>Reference to check.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| boolean | <p>True if <code>value</code> is an <code>Array</code>.</p>  |
