



# angular.lowercase








Converts the specified string to lowercase.







  

## Usage
```js
angular.lowercase(string);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| string | string | <p>String to be converted to lowercase.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| string | <p>Lowercased string.</p>  |
