



# angular.isDefined








Determines if a reference is defined.







  

## Usage
```js
angular.isDefined(value);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| value | * | <p>Reference to check.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| boolean | <p>True if <code>value</code> is defined.</p>  |
