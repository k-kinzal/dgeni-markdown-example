



# angular.uppercase








Converts the specified string to uppercase.







  

## Usage
```js
angular.uppercase(string);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| string | string | <p>String to be converted to uppercase.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| string | <p>Uppercased string.</p>  |
