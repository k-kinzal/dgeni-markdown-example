



# angular.fromJson








Deserializes a JSON string.







  

## Usage
```js
angular.fromJson(json);
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| json | string | <p>JSON string to deserialize.</p>  |

### Returns

| Type | Description |
| :--: | :--: |
| Object&#124;Array&#124;string&#124;number | <p>Deserialized thingy.</p>  |
