



# ngDblclick








This wrapped on `ngDblclick` directive allows you to specify custom behavior on a dblclick event on a fa-surface .








## Directive Info


* This directive executes at priority level 0.


## Usage


```
```html
<ANY ng-dblclick="expression">

</ANY>
```
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| ngDblclick | expression | <p>(Expression)[guide/expression] to evaluate upon a dblclick. (The Event object is available as <code>$event</code>)</p>  |
