



# faCanvasSurface








This directive creates a Famo.us CanvasSurface.








## Directive Info


* This directive executes at priority level 0.


## Usage


```
```html
<fa-canvas-surface fa-size="[400,400]">
</fa-canvas-surface>
```
```





### Arguments

| Param | Type | Details |
| :--: | :--: | :--: |
| faSize | String | <p>Array that passes width and height to the canvas</p>  |
